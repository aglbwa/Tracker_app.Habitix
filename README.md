
# 🎯 Tracker App

Мобильное приложение для отслеживания привычек и достижений, разработанное на Kotlin для Android.

## 📱 Функциональность

### 🎯 Основные возможности
- **Трекинг привычек** - создание и мониторинг ежедневных привычек
- **Система достижений** - получение наград за выполнение целей
- **Статистика** - детальная аналитика прогресса
- **Таймеры** - отслеживание времени выполнения задач
- **Профиль пользователя** - персонализация и настройки

### 👥 Администрирование
- **Панель администратора** - управление пользователями и данными
- **Авторизация** - безопасный вход в систему

## 🎨 Ключевые компоненты

### 🏆 Achievement System
- `Achievement` - модель достижения
- `AchievementManager` - управление логикой достижений
- `AchievementAdapter` - адаптер для списка достижений

### 💫 Habit Tracking
- `Habit` - модель привычки
- `HabitStatus` - статусы выполнения привычек
- `HabitAdapter` - отображение списка привычек

### 📊 Analytics
- `StatEntry` - запись статистики
- `StatsActivity` - экран с аналитикой

### ⏰ Time Management
- `TimerActivity` - управление таймерами
- `DeadlineWorker` - фоновые уведомления о дедлайнах

### 👤 User Management
- `User` - модель пользователя
- `ProfileActivity` - управление профилем
- `AuthActivity` - авторизация

### 🔧 Admin Features
- `AdminActivity` - панель администратора
- `DataRepository` - централизованное управление данными

## 🛠 Технологии

- **Язык**: Kotlin
- **Архитектура**: MVVM/MVI
- **Работа с данными**: Room Database, DataStore
- **Фоновые задачи**: WorkManager
- **Асинхронность**: Coroutines, Flow
- **DI**: Dagger Hilt / Koin

## 🔧 Настройка

### Конфигурация базы данных
Приложение использует Room для локального хранения данных. Все настройки находятся в `DataRepository`.

### WorkManager
`DeadlineWorker` отвечает за фоновые уведомления о приближающихся дедлайнах.
